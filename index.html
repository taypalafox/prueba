Template de pruebas

Agregar al repositorio

  • git init
  • git add .
  • git commit -m "base template"
  • git branch -M main
  • git remote add origin https://github.com/taypalafox/prueba.git
  • git push -u origin main
  • git pull --rebase origin main

  • git status
  • git add index.html

  • git push -u origin main
